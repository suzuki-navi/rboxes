office2pdf - Document to PDF converter

DESCRIPTION:
    Converts Microsoft Office documents to PDF format.

USAGE:
    office2pdf [options] <PATH>

OPTIONS:
    --help              Display this help message and exit

ARGUMENTS:
    <PATH>              Path to the office document to convert to PDF.
                        The output PDF will be created in the same directory
                        with the same filename but with .pdf extension.

SUPPORTED FORMATS:
    Input formats:
    - Word:            .doc, .docx
    - Excel:           .xls, .xlsx
    - PowerPoint:      .ppt, .pptx

    Output format:
    - PDF:             .pdf

EXAMPLES:
    office2pdf /path/to/document.docx
        Converts document.docx to document.docx.pdf

    office2pdf ~/report.xlsx
        Converts report.xlsx to report.xlsx.pdf

    office2pdf presentation.pptx
        Converts presentation.pptx to presentation.pptx.pdf

NOTE:
    The converted PDF file will be created alongside the original file
    with the original filename plus ".pdf" extension.
